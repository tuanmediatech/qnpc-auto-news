# PHẦN MỀM TỰ ĐỘNG LẤY TIN VỀ GOOGLE SHEETS - PC QUẢNG NGÃI

## CÁCH CHẠY PHẦN MỀM:
1. Mở thư mục "phanmem-qnpc"
2. Nhấn đúp chuột vào file: **run.bat** (hoặc `app-web-qnpc-fn.exe`)
3. Đợi khoảng 20 giây phần mềm sẽ tự động:
   - Lấy bài viết mới nhất
   - Ghi dữ liệu vào Google Sheets

## YÊU CẦU:
- Máy có kết nối internet
- Không cần cài Python
- Đã dán sẵn file: `my-project-xxx.json` (nếu cần thay đổi, liên hệ kỹ thuật)

## LỖI THƯỜNG GẶP:
- Lỗi trình duyệt: kiểm tra thư mục `pw-browsers` có đủ file
- Lỗi kết nối Google: kiểm tra file json đúng & có quyền chỉnh sửa Google Sheets
